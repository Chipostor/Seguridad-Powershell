# Seguridad-Powershell
Repositorio con scripts de ciberseguridad desarrollados en PowerShell orientados a automatización, recopilación de información del sistema y monitoreo mediante APIs.
El objetivo del repositorio es centralizar herramientas simples que ayuden en tareas de seguridad, auditoría y análisis de sistemas Windows.

Ejemplo:
Script que permite llamar APIs externas y registrar sus respuestas para monitoreo, auditoría o integración con herramientas de seguridad.
Script que recopila información del sistema local útil para auditorías de seguridad o inventario.

Mi objetivo es aprender a automatizar tareas con Powershell y ademas defender máquinas.
